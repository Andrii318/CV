Email Subscription & Dispatch Module

Backend-модуль для керування email-підписками, підтвердженням підписки, логікою відписки та масовими email-кампаніями.

Реалізовано

  • Підтвердження підписки через одноразовий код підтвердження.
  • Зберігання та валідація кодів підтвердження на базі Redis.
  • Активація підписок new_events і super_price.
  • Email-сповіщення про зміну статусу підписки.
  • Масова відправка листів підписникам.
  • Фільтрація контенту за мовою, містами, секціями та історією замовлень клієнта.
  • HTML email templates російською та івритом.
  • Логування відправки листів та обробка помилок.

Технології: Python, Flask, MySQL, Redis, HTML Templates, Email, REST API, JavaScript

Mail Analyzer / Bounce Email Processing

Модуль для аналізу bounce/DSN-повідомлень і виявлення email-адрес з проблемами доставки.

Реалізовано

  • Інтеграція Gmail API через OAuth.
  • Обробка вхідних email-повідомлень.
  • Парсинг MIME/DSN-структур і заголовків X-Failed-Recipients.
  • Витягування проблемних email-адрес із заголовків і тіла листа.
  • Обробка кейсів user unknown, mailbox unavailable, address not found, quarantine/spam, inactive email та changed email address.
  • Аналіз тексту листів на основі регулярних виразів.
  • Формування JSON-звітів для кандидатів на відписку. Використання OpenAI API як fallback для складного аналізу тексту листів.

Технології: Python, Gmail API, OAuth, Regex, MySQL, JSON, OpenAI API

Exim Mail Log Analyzer

Невеликий внутрішній CLI-інструмент для аналізу логів поштового сервера Exim. Проєкт ще в процесі розробки.

Реалізовано

  • Аналіз файлів mainlog і архівних mainlog.*.gz за вибраний період часу.
  • Пошук записів про failed і deferred доставку.
  • Витягування email-адрес із логів Exim.
  • Класифікація проблем: nonexistent user, unavailable mailbox, typo domain, temporary errors, блокування Spamhaus.
  • Генерація звітів .txt, .csv і summary.txt.
  • Генерація SQL import file для обробки відписки.

Технології: Python, Regex, CSV, Gzip, CLI, Exim logs, SQL
